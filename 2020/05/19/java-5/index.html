复制

所以两个对象指向的同一个实例

浅拷贝

浅拷贝就是将一个对象中的数据原封不动的全部拷贝给另一个对象（其实就是调用了Object类的默认的Clone方法），如果一个类中的数据都是基本类型或者不可变类的对象和话，这其实不会带来如何问题。
但是一个类如果含有可变对象的引用，这就会带来问题，两个对象中的对象的引用指向的是同一个对象，所以当一个对象调用了set之类的方法，另一个对象中的引用的值也就一起改变了。
