mysql 语法

1. 查看库的信息

一下是查看库数据的语法：
user databaseName使用某个数据库
show columns from tableName和desc tableName的作用都是查看表结构
show status用于显示广泛的服务器状态信息
show create database/table databaseName/tableName分别用来显示创建特定数据库或表的mysql语句
show grants用来显示授权用户(所有用户或特定用户)的安全权限
show errors/warnings用来显示服务器错误或警告消息
SELECT DISTINCT CONCAT('User: ''',user,'''@''',host,''';') AS query FROM mysql.user查看所有的用户对象
show grants for 'root'@'localhost'查看某一用户的权限

2. 检索数据语法

select */[columnName1,columnName2..] from tableName这是最简单的查询语句，无排序的，按照插入的顺序进行排序的数据
distinct关键字用以去除相同的值，这个关键字必须放在列名的最前面
select distinct username from user就是将user表中找到的所有username的重复值删去
limit number限制显示的行数
limit number1,number2 或 limit number2 offset number1显示从number1开始的number2行，number1默认0开始
select tableName.columnName from tableName可以使用全限定名

3. 对数据进行排序

order by选择一个或者多个列，进行排序
desc使用改关键字实现降序，一个desc只对其前面的关键字有效，若多列都想实现降序，每个排序的列后都需要添加desc关键字

4.数据的过滤

where我们在使用where时在from之后在order by之前
<> != = < > <= >= between这是where中所有的条件操作符
where columnName is null在使用空值查询的时候使用的是is
column1 < 100 and/or column2 >200使用and或or进行组合查询，但是在大量的and和or操作组合的场景中，优先处理and关键字，为了避免出现预期之外的组合，建议使用圆括号对条件进行组合
in在使用in关键字的时候非常灵活，执行的速度快于or，并且还可以进行子查询
not否定后面条件的关键字

4.1使用通配符过滤

name like 'A%'意味以A开头的所有名字，有一点需要注意的是’%’不能用于匹配null
name like '_A%'这个通配符标识一个位子，即第二位为A的所有名字
有一点需要注意的是通配符在搜索的时候性能是会降低的，实际中可以不使用通配符尽量减少使用

4.2正则表达式的使用

正则表达式的作用是将正则表达式和文本串进行比较
selecct columnName from tableName where columnName regexp '正则表达式'使用关键字 regexp，正常情况下是不区分大小写的可以使用regexp binary进行区分大小写
'number1|number2'正则中使用or进行匹配的方式
[123] Ton[]中的内容是或的意思，只取其中的一个字符，就是匹配1 Ton、2 Ton和3 Ton
.是匹配任意值
还可以匹配字符类[:alnum:]等等

5.数据拼接

select concat(username,'(',sex,')') as look from user可以对自己想要查询的数据进行拼接
在mysql中也可以直接使用计算

6.函数

6.1文本处理函数

rtrim()去除串尾空格
upper()将文本转换为大写
left()返回串左边的字符
length()返回串的长度
locate()找出串的一个子串
lower()将串转换为小写
ltrim()去掉串左边的空格
right()返回串右边的字符
soundex()返回串的soundex值
substring()返回字串的字符

6.2日期和时间处理函数

AddDate()增加一个日期（天、周等）
AddTime ()增加一个时间（时、分等）
CurDate ()返回当前日期
CurTime ()返回当前时间
Date()返回日期时间的日期部分
DateDiff()计算两个日期之差
Date_Add()高度灵活的日期运算函数
Date_Format()返回一个格式化的日期或时间串
Day()返回一个日期的天数部分
DayOfWeek()对于一个日期，返回对应的星期几
Hour()返回一个时间的小时部分
Minute()返回一个时间的分钟部分
Month()返回一个日期的月份部分
Now()返回当前日期和时间
Second()返回一个时间的秒部分
Time()返回一个日期时间的时间部分
Year()返回一个日期的年份部分

6.3数值处理函数

Abs()返回一个数的绝对值
Cos()返回一个角度的余弦
Exp()返回一个数的指数值
Mod()返回除操作的余数
Pi()返回圆周率
Rand()返回一个随机数
sin()返回一个角度的正弦
Sqrt()返回一个数的平方根
Tan()返回一个角度的正切

6.4聚集函数

avg()返回某列的平均值
count()返回某列的行数
max()返回某列的最大值
min()返回某列的最小值
sum()返回某列值之和
distinct在计算值的时候使用distinct是使用不同的值

8.主键以及外键

8.1 主键

主键的作用是可以在一个表中唯一标识一个一行数据

8.2 外键

可以唯一标识关联的表的数据

联结

table1,table2这种联结的结果是返回笛卡儿积集
可以使用where语句进行限定
inner join,left join,right join,nature join这四种join在使用的过程中可以通过on关键字进行限定，大多数情况下自然连接(nature join)数据库是未实现的，需要我们自己动手将重复的列去除

内部联结
内连接（等值连接）
自然连接（特殊的内连接：去除进行比值的同一列）
外部联结（在使用外部连接的时候必须具有on语句）
左外连接：右侧的表中的值可以为空
右外连接：左侧的表中的值可以为空

9.组合查询

union在两个查询中间加上一个union
union连接的两个查询需要具有相同的列、表达式或聚集函数
union会自动删除重复的行
可以使用union all将所有的行（包含重复的）输出
在使用组合查询的语句中order by语句只能出现在最后一个查询语句中使得排序统一
